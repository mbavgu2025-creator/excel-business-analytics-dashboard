## Project Overview
This project presents an Excel-based business analytics dashboard created using Vrinda Store sales data to analyze overall business performance and customer behavior.

## Dataset
- Vrinda Store Sales Data (2024)

## Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP / XLOOKUP
- Charts & Data Visualization

## Key Analysis Performed
- Sales vs Orders performance analysis
- Men vs Women sales comparison
- Age vs Gender customer analysis
- Order status tracking
- Top 5 states by sales
- Order channel performance analysis

## Insights Generated
- Identified key customer segments by gender and age
- Analyzed sales contribution across different states
- Evaluated order fulfillment status
- Compared performance across sales channels

## Purpose
This dashboard was created as part of MBA Business Analytics learning to understand business data analysis and insight generation using Excel.
